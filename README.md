# Kiran Yadav Portfolio

A personal portfolio website for Kiran Yadav, focused on backend engineering, AI agents, automation, and scalable systems.

## Features

- Hero section with profile image and intro
- About section with profile summary
- Skills, experience, projects, education, and contact sections
- Responsive layout for desktop and mobile

## Tech Stack

- HTML
- CSS
- JavaScript
- Lucide icons
- Google Fonts

## Local Run

Open `index.html` directly in a browser, or use a simple local server.

Example:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Deploy to GitHub

1. Create a GitHub repository.
2. Push this project to the repo.
3. Enable GitHub Pages in repository settings.
4. Select the `main` branch and `/root` folder.

Your site will then be accessible from anywhere through the GitHub Pages URL.
